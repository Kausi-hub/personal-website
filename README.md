# Personal Portfolio Website\n\nA modern, performant portfolio website showcasing expertise in systems engineering, controls, and autonomous motion control.\n\n## 🚀 Tech Stack\n\n- **Framework**: Next.js 15\n- **Language**: TypeScript\n- **Styling**: Tailwind CSS + shadcn/ui\n- **Animations**: Framer Motion\n- **Icons**: Lucide React\n- **Content**: MDX (Markdown + JSX)\n- **Hosting**: Vercel\n- **CI/CD**: GitHub Actions\n\n## 📁 Project Structure\n\n```\n├── src/\n│   ├── app/              # Next.js app router pages\n│   ├── components/       # Reusable React components\n│   ├── lib/              # Utility functions\n│   └── content/          # MDX blog posts and content\n├── public/               # Static assets\n├── .github/workflows/    # CI/CD workflows\n└── content/              # MDX content files\n```\n\n## 🛠️ Getting Started\n\n### Prerequisites\n\n- Node.js 18+\n- npm, yarn, or pnpm\n\n### Installation\n\n```bash\ngit clone https://github.com/Kausi-hub/personal-website.git\ncd personal-website\nnpm install\ncp .env.example .env.local\n```\n\n### Development\n\n```bash\nnpm run dev\n```\n\nOpen http://localhost:3000 in your browser.\n\n### Build & Deploy\n\n```bash\nnpm run build\nnpm start\nnpm run format\nnpm run type-check\n```\n\n## 📄 Site Navigation\n\n- **Home** - Landing page with hero section and expertise highlights\n- **About** - Professional background, leadership experience, and detailed expertise\n- **Expertise** - Systems Engineering, Controls, System Integration, V&V\n- **Programs** - EDU Platform, GM RWD, AMG eLSD, AWD Systems, Autonomous Motion Control\n- **Innovation** - Patents and technical contributions\n- **Blog** - Technical insights and articles (MDX-based)\n- **Contact** - Contact form for reaching out\n- **Resume Center** - Download and view resume\n- **Career Timeline** - Interactive career progression\n\n## 🚀 Deployment\n\nThis project is configured for Vercel deployment. Connect your GitHub repository to Vercel for automatic deployments on push to main branch.\n\n## 📝 License\n\nMIT License\n
